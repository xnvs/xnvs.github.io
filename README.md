Книги:

[Алгоритмы на Java. Современный подход.](https://xnvs.github.io/java-algorithms-nizov/) 
pdf версия для скачивания внутри.

Мы будем стремиться сделать полный и исчерпывающий курс по алгоритмам на Java, 
от разбора основных тем и прохождения технических интервью до специализаций.

### Что уже реализовано:
+ все основные типы алгоритмов и структур данных
+ связанные с алгоритмами soft skills 
+ полный гайд по traversal по деревьям и матрицам
+ пятьдесят основных задач в решебнике (полный разбор) + два десятка в основной книге

### Что планируется к добавлению на конец 2026 начало 2027:
+ полный гайд по математике и математической нотации (как разбирать мат нотацию алгоритмов) + 150 форматных страниц
+ приложение по типам алгоритмов и их математике (основная книга форматно не включает математику в качестве объяснения) + 50 страниц
+ дополнительные пятьдесят популярных задач, минимум половина категории hard + 100 страниц

### Что планируется к добавлению перспективно (не в следующей итерации):
+ отдельный раздел: алгоритмы специфики геймдева (pathfinding, сейчас лишь A*, коллизии, AILogic, PCG)
+ отдельный раздел: финансовые и стат алгоритмы + основные подходы в обработке данных
+ отдельный раздел: работа с tensorflow через Java
+ отдельный раздел: алгоритмы для базового программирования нейросетей с нуля
